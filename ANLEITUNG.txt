==========================================================
  FINANZEN & VERMÖGEN  –  Anleitung
==========================================================

LIVE-APP: https://steffenah.github.io/Finanzen/

==========================================================
  ERSTEINRICHTUNG (einmalig, ~5 Minuten)
==========================================================

SCHRITT 1 — Datenbank anlegen (Supabase, kostenlos)
---------------------------------------------------
1. Auf supabase.com kostenlos registrieren → "New Project".
2. Im Projekt links auf "SQL Editor" → "+ New query".
3. In der App auf dem Einrichtungsbildschirm den Knopf
   "SQL kopieren" → Code in Supabase einfügen → "Run".
   ("Success. No rows returned" = alles gut)
4. Im Supabase-Projekt links unter "Project Settings":
   - "General"  → "Project ID" kopieren.
   - "API"      → "anon public" Key kopieren.
5. Beides in der App eintragen → "Verbinden".
6. Beim ersten Login ein eigenes Passwort festlegen (merken!).

SCHRITT 2 — Aktien-Kurse abrufen (optional)
-------------------------------------------
1. Gratis-API-Key holen: alphavantage.co/support/#api-key
2. In der App: "Einstellungen" → Alpha-Vantage-Key eintragen.
   (~25 Abrufe/Tag kostenlos — für die monatliche Prüfung reicht das.)
   Ticker-Symbol-Beispiele:  US-Börse → AAPL,  Xetra → SAP.DE


==========================================================
  AUF WEITERE GERÄTE ÜBERTRAGEN (Handy / 2. PC)
==========================================================

Einfacher Weg:
  Einstellungen → "Auf weiteres Gerät übertragen" →
  "Setup-Link erzeugen" → Link kopieren ODER QR scannen.
  Auf dem neuen Gerät öffnen → fertig, nur noch Passwort eingeben.

Als App installieren:
  - iPhone (Safari):  Teilen → "Zum Home-Bildschirm"
  - Android (Chrome): Menü → "App installieren"
  - Oder den 📲-Knopf unten rechts in der App nutzen.


==========================================================
  CODE-UPDATES (für Entwickler-Modus)
==========================================================

"start.ps1" doppelklicken:
  - holt aktuelle Code-Version von GitHub (pull)
  - lädt eigene Änderungen automatisch hoch (push)

Wichtig: Immer nur auf EINEM Gerät zur gleichen Zeit Code ändern
und start.ps1 zu Ende laufen lassen, bevor du auf das andere Gerät
wechselst.

Hinweis: Diese Updates betreffen NUR den App-Code. Deine Finanzdaten
liegen IMMER in Supabase und sind auf allen Geräten automatisch
synchron, sobald die Verbindung steht.


==========================================================
  BACKUP & WIEDERHERSTELLEN
==========================================================

Einstellungen → "Backup herunterladen" → speichert alle Daten
als JSON-Datei auf deinem Rechner.

"Backup wiederherstellen" → liest eine JSON-Datei zurück
(überschreibt die Cloud-Daten — bitte vorsichtig).

Empfehlung: alle 1–2 Monate ein Backup ziehen.


==========================================================
  WICHTIGER HINWEIS
==========================================================
Die Steuer- und Netto-Berechnungen sind unverbindliche Schätzungen
(keine Steuerberatung). Die Vorabpauschale auf thesaurierende Fonds
ist nicht enthalten. Bank-Zinssätze sind Vorschläge und sollten bei
Bedarf angepasst werden. Die Anlage-KAP-Werte vor Steuererklärung
bitte mit den Jahressteuerbescheinigungen der Banken abgleichen.
