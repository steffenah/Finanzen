☁️ Einmalige Einrichtung

Am schnellsten geht's mit einem Setup-Link von einem Gerät, auf dem die App bereits läuft — kein Tippen, keine Codes.

  1. Auf dem eingerichteten Gerät (PC/Laptop): App öffnen → ⚙️ Einstellungen.
  2. Button 🔗 Setup-Link erzeugen klicken.
  3. QR-Code mit diesem Gerät scannen — oder Link per WhatsApp/Mail schicken und hier öffnen.

→ Verbindung ist danach automatisch eingerichtet. Du musst nur noch dein Passwort eingeben.

Du brauchst dafür eine kostenlose Supabase-Datenbank:

  1. Auf supabase.com kostenlos registrieren → New Project anlegen.
  2. Links im Menü auf SQL Editor → unten stehenden Code einfügen → Run.
  3. Project Settings → General: Project ID kopieren. → API / API Keys: anon public key kopieren. Beides unten einsetzen.
Findest du unter Settings → General. Die App baut die Adresse https://[ID].supabase.co automatisch. (Eine komplette URL geht auch.)

💵 Dividenden & Prognose

📈 Prognose: Aus den erfassten Dividenden-Jahren (bis zu 10) wird ein Durchschnitt gebildet und als erwartete Jahres-Dividende in die Zukunft fortgeschrieben.

🧾 Steuer-Schätzung

⚠️ Unverbindliche Schätzung der deutschen Abgeltungsteuer (25 % + Soli + ggf. Kirchensteuer) auf Zins- und Dividendenerträge. Vorabpauschale auf thesaurierende Fonds ist nicht enthalten. Keine Steuerberatung.

📑 Anlage KAP — Vorausfüllhilfe

⚠️ Schätzung anhand deiner App-Daten (Zinssätze × Bestände, erfasste Dividenden). Vor der Steuererklärung mit der Jahressteuerbescheinigung deiner Bank abgleichen!

📊 Vermögens-Verlauf

📅 Bei jedem Monatswechsel speichert die App automatisch einen Snapshot. So siehst du, wie sich euer Vermögen über die Zeit entwickelt — gerade während des Hausbaus motivierend.

📈 Jährliche Anpassung (Inflation & Gehalt)

Wird jedes Jahr auf fixe Ausgaben und auf das Netto-Einkommen angewandt. So bleibt die reale Kaufkraft in der Prognose etwa konstant. Typisch: 2 – 3 %. Auf 0 setzen für keine Anpassung.

Automatischer Kurs-/Dividenden-Abruf

Kostenlosen API-Schlüssel bei alphavantage.co/support/#api-key holen (gratis, ohne Kreditkarte). Limit: ~25 Abrufe/Tag — für eine monatliche Prüfung reicht das.

🔢 PIN-Schnellzugriff (nur auf diesem Gerät)

Kein PIN gesetzt — du musst dich mit Passwort anmelden.

Die PIN wird nur lokal in diesem Browser gespeichert (nicht in der Cloud). Auf jedem Gerät kannst du eine eigene PIN setzen.

💾 Datensicherung

Alle Daten (Konten, Wertpapiere, Einstellungen, Hausbau, Kredite, Verlauf) als JSON-Datei sichern. Empfehlung: ab und zu eine Sicherung herunterladen — falls Supabase mal Probleme macht oder du den Account neu aufsetzt.

📲 Auf weiteres Gerät übertragen

Du sparst dir die manuelle Eingabe von Project ID + Key auf dem Handy: Erzeuge unten einen Setup-Link, schick ihn dir aufs Handy (WhatsApp / Mail) oder scan den QR-Code. Beim Öffnen ist die Verbindung dort sofort eingerichtet — du musst nur noch dein Passwort eingeben.

⚠️ Behandle den Link wie ein Passwort — wer ihn hat, kommt an deine Daten (sofern er das App-Passwort kennt).
